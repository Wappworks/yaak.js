Introduction

YAAK (Yet Another Audio Kit) is an Open Source HTML5 game audio system for web browsers and native HTML5 app acceleration platforms such as CocoonJs and AppMobi. The library is hosted on GitHub and is distributed under the FreeBSD Open source license.

Click here for the GitHub repository

System Features

  • Audio loading/caching
  • Mute/unmute
  • Standard sound playback and looping
  • Sound channel support
    • Polyphony limits
    • Volume control
    • Mute/un-mute
    • Stop all sounds
  • Per active sound playback control
    • Stop
    • Volume control
